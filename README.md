# Project Skeleton

A Skeleton project for working on

- PHP7
- Phpunit 6
- Docker

## Start the project

    $ composer create-project malotor/skeleton my_project dev-master
    $ cd my_project
    $ docker-compose build
    $ docker-compose run composer
    $ docker-compose up -d app
    
## Run the tests

    $ docker-composer run phpunit
    
## App

http://localhost:8080
